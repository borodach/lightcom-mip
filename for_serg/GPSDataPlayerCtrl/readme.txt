Не блокирует контролы, при добавлении в использующие приложения.

В следствии не блокирования контролов при не загруженных GPS данных, происходит сбой при попытке
манипулировать элементами управления проигрывателя.

Не реализована возможность читать данные с сервера на прямую (радио).

Не эргономичный интерфейс.

Возможно есть дополнительные ньюансы которые мной не были выявлены.

Функционал GPSDataPlayer насколько я помню полностью реализован.
