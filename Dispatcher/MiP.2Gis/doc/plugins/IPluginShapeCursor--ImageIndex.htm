IPluginShapeCursor::ImageIndex

Материал из 2gis-SDK.

Перейти к: навигация, поиск

Получение индекса рисунка в коллекции по заданному типу.

Спецификация IDL
[
    propget
]
HRESULT ImageIndex( [in] enum SymbolImageType enImageType, [out, retval] long *pIndex );
Параметры
enImageType Тип SymbolImageType рисунка
pIndex Возвращаемое значение, индекс рисунка в коллекции

Возвращаемые значения
S_OK Значение возвращено
S_FALSE В случае неудачи
