2gis-SDK:Библиотека типов GrymCore

Материал из 2gis-SDK.

Перейти к: навигация, поиск

COM-библиотека типов Grym Core Library декларирует открытые интерфейсы API ДубльГИС. Библиотека включена в главный модуль приложения Grym.exe и регистрируется в операционной системе при первом запуске приложения. После запуска приложения библиотека типов становится доступной для импорта. Ниже приведена спецификация IDL библиотеки и полный алфавитный перечень деклараций всех типов, в нее входящих. В панели "справочник" расположен перечень интерфейсов библиотеки, разбитых на функциональные категории.

	uuid(7AA02C95-0B4A-43aa-92D8-BA40511A7F3F),

	helpstring("GrymCore 1.0 Type Library")
