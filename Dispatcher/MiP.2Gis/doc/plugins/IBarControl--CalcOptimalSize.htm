IBarControl::CalcOptimalSize

Материал из 2gis-SDK.

Перейти к: навигация, поиск

Метод служит для пересчета контролом оптимального размера.

Спецификация IDL
HRESULT CalcOptimalSize([in] LONG nBarInteriorHeight, [in] IDevSize *pMaxSize, 
    [out, retval] IDevSize **pVal);
Параметры
nBarInteriorHeight размер по-вертикали клиентской части родительской панели
nMaxSize Указатель на интерфейс IDevSize, максимальный размер контрола
pVal Возвращаемое значение, указатель на интерфейс IDevSize, оптимальный размер контрола

Возвращаемые значения
S_OK Интерфейс возвращен
S_FALSE В случае неудачи


Смотри также: IBarControl::MinHeight, IBarControl::CalcWidth
