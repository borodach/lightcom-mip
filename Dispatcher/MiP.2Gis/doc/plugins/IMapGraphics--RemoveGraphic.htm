IMapGraphics::RemoveGraphic

Материал из 2gis-SDK.

Перейти к: навигация, поиск

Удаление пользовательского графического объекта из карты.

Спецификация IDL
HRESULT RemoveGraphic([in] IGraphicBase *pGraphic);
Параметры
pGraphic Интерфейс IGraphicBase пользовательского графического объекта

Возвращаемые значения
S_OK Объект удален
S_FALSE В случае неудачи
